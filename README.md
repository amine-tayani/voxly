# Voxly

### A full stack location based question and answers.

### Clone Repository

```
git clone https://github.com/amine-tayani/voxly.git
```

### Client Installation

```
cd frontend
npm install
npm run dev
```

## Server Installation

```
cd server
npm install
```

#### Add .env file to root server directory

```
PORT=
MONGODB_URL=
JWT_SECRET=
SMTP_HOST=
SMTP_PORT=
SMTP_USER=
SMTP_PASSWORD=
SMTP_FROM=
COOKIE_SECRET=thisisasamplesecret
CLIENT_URL=http://localhost:3000

```
